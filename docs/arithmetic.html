Arithmetic Module

The Arithmetic module provides fundamental mathematical operations used throughout TheiaMath.

It includes addition, subtraction, multiplication, division, modulo operations, and integer variants.

Importing the Module

import theia.math.Arithmetic;

Import the Arithmetic class before using its methods.

Addition

double result = Arithmetic.Add(10, 5);

System.out.println(result);

15.0

Returns the sum of two numbers.

Subtraction

double result = Arithmetic.Sub(10, 5);

System.out.println(result);

5.0

Returns the difference between two numbers.

Multiplication

double result = Arithmetic.Mul(10, 5);

System.out.println(result);

50.0

Returns the product of two numbers.

Division

double result = Arithmetic.Div(10, 5);

System.out.println(result);

2.0

Returns the quotient of two numbers.

Division by Zero

Arithmetic.Div(10, 0);

Throws an ArithmeticException.

Modulo

double result = Arithmetic.Mod(10, 3);

System.out.println(result);

1.0

Returns the remainder of a division.

Integer Division

int result = Arithmetic.Idiv(10, 3);

System.out.println(result);

3

Performs division and returns an integer result.

Integer Modulo

int result = Arithmetic.Imod(10, 3);

System.out.println(result);

1

Performs modulo and returns an integer result.

Method Reference

Method Description
Add(a, b) Adds two values.
Sub(a, b) Subtracts two values.
Mul(a, b) Multiplies two values.
Div(a, b) Divides two values.
Mod(a, b) Returns the remainder.
Idiv(a, b) Integer division.
Imod(a, b) Integer remainder.

Example Program

import theia.math.Arithmetic;

public class Main {

    public static void main(String[] args) {

        System.out.println(
            Arithmetic.Add(5, 3)
        );

        System.out.println(
            Arithmetic.Sub(5, 3)
        );

        System.out.println(
            Arithmetic.Mul(5, 3)
        );

        System.out.println(
            Arithmetic.Div(5, 3)
        );

    }

}
