Getting Started

Welcome to TheiaMath.

This guide will help you create your first project and perform basic mathematical operations using the library.

Before You Begin

Make sure that:

  • Java 8 or later is installed.
  • TheiaMath has been downloaded.
  • The library has been added to your project.

If you have not completed these steps, visit the Installation Guide first.

Open Installation Guide

Your First Program

Create a file named: 

Main.java

Then write the following code: 

import theia.math.Arithmetic;

public class Main {

    public static void main(String[] args) {

        double result = Arithmetic.Add(5, 3);

        System.out.println(result);

    }

}

Compile the Program

javac -cp lib/TheiaMath.jar Main.java

This command compiles your Java source file while including TheiaMath in the classpath.

Run the Program

java -cp .:lib/TheiaMath.jar Main

Expected output: 

8.0

Basic Operations

Addition

Arithmetic.Add(10, 5);

Subtraction

Arithmetic.Sub(10, 5);

Multiplication

Arithmetic.Mul(10, 5);

Division

Arithmetic.Div(10, 5);

Modulo

Arithmetic.Mod(10, 3);

Working with Variables

double a = 15;
double b = 4;

double result = Arithmetic.Div(a, b);

System.out.println(result);

Using variables makes your programs more flexible and easier to maintain.

Handling Errors

Division by zero is not allowed. 

Arithmetic.Div(10, 0);

The code above will throw an ArithmeticException.
