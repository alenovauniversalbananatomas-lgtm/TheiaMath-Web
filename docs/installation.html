Installation

This guide explains how to install TheiaMath and add it to your Java projects.

Requirements

  • Java 8 or later
  • A Java compiler (javac)
  • TheiaMath JAR file

Download

Download the latest release from the Downloads page.

Open Downloads

Add the JAR to Your Project

Place the TheiaMath JAR file inside your project's library directory. 

project/
│
├── lib/
│   └── TheiaMath-v1.0.0.jar
│
└── src/

Compile

Compile your project using the JAR in the classpath. 

javac -cp lib/TheiaMath-v1.0.0.jar Main.java

Run

Run the application with TheiaMath included in the classpath. 

java -cp .:lib/TheiaMath-v1.0.0.jar Main

First Program

import theia.math.Arithmetic;

public class Main {

    public static void main(String[] args) {

        double result = Arithmetic.Add(5, 3);

        System.out.println(result);

    }

}

Expected output: 

8.0
