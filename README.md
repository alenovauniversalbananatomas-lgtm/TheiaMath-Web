# TheiaMath Website

Official website of TheiaMath.

This repository contains the source code of the official TheiaMath website, including documentation, downloads, examples, release information, and project resources.

## About TheiaMath

TheiaMath is a lightweight and high-precision mathematics library for Java designed to provide arithmetic, algebra, geometry, mathematical constants, and future advanced mathematical functionality.
